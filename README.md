#Spotify Stats - SQL & Tableau

##Overview
This project analyzes personal Spotify data using **MySQL** for data storage and queries, and **Tableau** for data visualization.
It explores song trends, artist performance, and listening behaviours and preferences.

---

##Database Structure
- **Songs** - Song Details (Song Name, Album, Artist, Release Date, Song Length, Times Streamed)
- **Artists** - Artist Info (Artist Name, Country, Gender, Followers)
- **FemaleKoreanArtists (View)** - Summary of Female Korean Artists (Artist Name, Total Song Length, Total Streams, Followers)
